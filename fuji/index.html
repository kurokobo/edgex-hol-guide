    ハンズオンラボ for Fuji release¶

    Fuji リリース向けのハンズオンガイドです。

    Ubuntu 上の Docker 環境を前提に、セットアップから起動確認、デバイスサービスの構成とエクスポートまでを扱っています。

    Fuji リリースの扱い

    2020 年 4 月にも最新バージョンの Geneva のリリースが予定されています。Fuji からの変更点は非常に多いですが、全体の思想やアーキテクチャにはそこまでドラスティックな変革は入っていないように見えます。

    Fuji リリースに触れておけば、Geneva リリースの習得もしやすくなるはずです。

    必要なもの¶

    このガイドでは、以下の環境を前提としています。この環境自体の構築はこのガイドでは取り扱わないので、それぞれのドキュメントなどを参考に導入してください。

    • Ubuntu Desktop 18.04 LTS
      • Ubuntu Installation Guide

    仮想マシンでも構いませんし、Raspberry Pi などで直接動作させた環境でも問題ありません。

    また、この Ubuntu 環境には、以下を導入済みであるものとします。

    • Docker xx.xx
      • Get Docker Engine - Community for Ubuntu
    • Docker Compose xx.xx
      • Install Docker Compose

    この Ubuntu 環境は、以下の条件を満たす必要があります。

    • インタネットに接続できること
    • git コマンドが利用できること

    また、ラボには Web GUI や REST API の操作などを含むので、この Ubuntu 環境にアクセスできる端末で以下が利用できる状態にしておくとスムーズです。

    • Web ブラウザ
      • Google Chrome など
    • Postman
      • Postman | The Collaboration Platform for API Development

    Note

    ラボは基本的には Docker コンテナで完結するので、Windows でもほとんど同様に進められます。ただし、シェル環境の差異などもあるため、適宜読み替えは必要です。
