    EdgeX Foundry ハンズオンラボ¶

    EdgeX Foundry のハンズオンラボのガイドです。自分のペースで進められるセルフペースドラボ形式です。

    このガイドについて¶

    自分のペースで進められるセルフペースドラボ形式のため、時間や場所を問わず取り組んでいただけますが、主として EdgeX Foundry Tokyo のイベント向けに作成しています。

    必要なもの¶

    実際に環境を作りながら進めるため、

    • Docker が動作する Ubuntu Desktop（または Ubuntu Server）環境

    が必要です。仮想マシンでも構いませんし、Raspberry Pi などでも構いません。

    ガイドで前提としている環境の詳細は、各ガイドの冒頭で説明しています。
